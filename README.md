## 📋 Commands

### 🌍 Public Commands

| Command | Description | Example |
|---------|-------------|---------|
| `.menu` | Show beautiful menu | `.menu` |
| `.news` | Fetch latest news | `.news` |
| `.stats` | Bot statistics | `.stats` |
| `.save` | Save replied media | `.save` |
| `.vv` | Save view-once media | `.vv` |
| `.status` | Status saver info | `.status` |
| `.admins` | List group admins | `.admins` |
| `.groupinfo` | Group details | `.groupinfo` |
| `.tagall` | Mention all members | `.tagall` |
| `.poll` | Create a poll | `.poll Best news source?` |
| `.afk` | Set AFK status | `.afk Sleeping` |

### 🛡️ Admin Commands

| Command | Description |
|---------|-------------|
| `.mute` / `.unmute` | Toggle group mute (30min) |
| `.warn @user` | Warn a member (3 = kick) |
| `.kick @user` | Remove a member |
| `.add 94xxxxx` | Add a member |
| `.promote @user` | Make admin |
| `.demote @user` | Remove admin |
| `.voice on/off` | Toggle voice replies |
| `.antilink on/off` | Toggle link protection |
| `.welcome on/off` | Toggle welcome message |
| `.goodbye on/off` | Toggle goodbye message |
| `.buttons on/off` | Toggle button menu |

### 👑 Owner Commands

| Command | Description |
|---------|-------------|
| `.settings` | View all settings |
| `.mode public/private/inbox/groups` | Change bot mode |
| `.autostatus on/off` | Toggle auto status |
| `.autonews on/off` | Toggle auto news |
| `.autobio on/off` | Toggle auto bio |
| `.setprefix !` | Change prefix |
| `.broadcast msg` | Send to all groups |
| `.ban @user` | Ban a user |
| `.unban @user` | Unban a user |
| `.banlist` | Show banned users |

## 📦 Local Setup

```bash
# 1. Clone the repository
git clone https://github.com/charukamahesh922-collab/NewsBot-LK.git

# 2. Navigate to project
cd NewsBot-LK

# 3. Install dependencies
npm install

# 4. Edit config.js
# - Add your WhatsApp number to ownerNumber
# - Add your group JID to newsGroupJid

# 5. Start the bot
npm start

# 6. Scan QR code with WhatsApp
