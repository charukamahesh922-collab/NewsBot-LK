# 📰 NewsBot-LK

WhatsApp News Bot for Sri Lankan News Sources

## 📡 Sources
- 🇱🇰 Hiru News (6 categories)
- 🔴 Derana News (Hot News)
- 🏏🇱🇰 Sinhala Cricket News
- 🏏🌍 English Cricket News (ESPN Cricinfo)

## ⚡ Features
- Auto-fetch every 2 minutes
- Full descriptions
- Images with news
- Auto-react emojis
- Bot menu commands
